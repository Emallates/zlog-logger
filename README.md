## Prerequisites
Clients should be registered with the regarding service to use log service

## Installation

Install stable version from NPM:
```
npm install zlog-logger --save
```


## Configuration

You have to set the mode and proide the api information.

#### Logger && Log

```javascript
//express
app.use(logger(conf))
//manual use
logger(//params);
```
#### Log using service

```javascript
//config
var conf = {
       appId:'app_id',
       apiKey:'api_key',
       ip:'host', port:'port',
       mode : 'central', || leave mode 
} 
app.use(logger(conf))

```
#### Log using local file

```javascript
//config
var conf = {
       appId:'app_id',
       apiKey:'api_key',
       ip:'host', port:'port',
       mode : 'local', || leave mode 
}
app.use(logger(conf))

```
